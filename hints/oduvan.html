Como elevar um número a uma potência em Python?

Você pode usar o operador de potência **: resultado = base ** expoente

Como lidar com índices fora do alcance?

Para verificar se um índice está dentro do alcance da lista, você pode usar uma condição: 

lista = [10, 20, 30]
N = 2
if N < len(lista):
    valor = lista[N]
else:
    valor = -1
Isso ajuda a evitar erros ao acessar índices que não existem na lista.
